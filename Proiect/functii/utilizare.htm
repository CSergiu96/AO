Utilizare

Introducere Circuite Ecuatii Utilizare

      Programul implementeaza amplificatoare operationale cu reactie negativa.

      Prima fereastra contine o poza reprezentativa a unui circuit integrat ce contine amplificatoare si doua butoane : START si DOCUMENTATIE :

  •       Butonul de START duce la urmatoarea pagina, cea in care se poate observa functionarea amplificatoarelor
  •       Butonul de DOCUMENTATIE deschide o pagina web (cea curenta) in care puteti afla mai multe despre proiect

      Interfata permite utilizatorului vizualizarea a patru circuite, introducerea de date si afisarea tensiunilor de intrare si iesire, pe baza datelor de intrare.

      Interfata e divizata in patru parti :

  •       Partea de date de intrare (dreapta jos) : are tre diviziuni
  1. Tip – Se selecteaza tipul de amplificator dorit
  2. Proprietati fizice -- R1 si R2 reprezinta cele doua rezistente ale amplificatoarelor ce pot fi modificate (***La repetor inversor rezistentele sunt identice si notate cu „R” iar la repetor neinversor nu este cazul), Iar Amplificarea este data in functie de cele doua rezistente
  3. Semnal de intrare – Modelarea semnalului de intrare in functie de parametrii acestuia
  •       Semnalul de intrare dat de utilizator (stanga sus)
  •       Semnalul de iseire (stanga jos)
  •        Schema circuitului selectat (dreapta sus)

      Pentru a face mai usoara utilizarea interfetei graficele si schema electrica o sa apara doar dupa selectarea circuitului

      La fiecare schimbare a oricarui editbox interfata se reactualizeaza si isi pastreaza valorile anterioare pentru celelalte editbox-uri
